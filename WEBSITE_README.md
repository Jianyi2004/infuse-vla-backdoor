# INFUSE Project Website

This is the project website for the paper "Inject Once Survive Later: Backdooring Vision-Language-Action Models to Persist Through Downstream Fine-tuning" (CVPR 2026).

## Website URL
https://jianyi2004.github.io/infuse-vla-backdoor/

## What's Included

### Completed Sections:
1. **Hero Section** - Paper title, authors, and affiliations
2. **Abstract** - Full paper abstract with key findings
3. **Method Overview** - Visual representation of the INFUSE pipeline
4. **Key Results** - Image carousel showing:
   - Attack persistence comparison
   - Module sensitivity analysis
   - Attention visualization
   - Trajectory comparisons
5. **Method Details** - Three-stage approach explanation
6. **Real-world Experiments** - Results on physical robot tasks
7. **Key Contributions** - Main contributions of the work
8. **BibTeX Citation** - Ready-to-copy citation

### Resources:
- All figures from the paper converted to high-resolution PNG (300 DPI)
- Responsive design that works on mobile and desktop
- SEO optimized with proper meta tags
- Structured data for academic search engines

## Next Steps:

### Before Publishing:
1. **Add Paper PDF**: Place your paper PDF in `static/pdfs/paper.pdf`
2. **Update arXiv Link**: Once paper is on arXiv, update the arXiv link in index.html (currently shows "Coming Soon")
3. **Add Code Repository**: When code is ready, update the GitHub link
4. **Optional**: Add demo videos to `static/videos/` folder

### To Deploy on GitHub Pages:
1. Push this repository to GitHub
2. Go to repository Settings → Pages
3. Select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Save and wait for deployment
